说明

easy-bootstrap 是一整套运用于CMS类WEB项目，JQuery 插件集合。包含了datagrid、tree、treegrid、window、tabpage、input等常用插件。

1.初始化方式


            $("#id").TreeGrid({
                prop:"prop",
                event:function(row){}
            });

2.调用方法


            var grid = $("#id").data("treegrid");
            var row = grid.getSelected();
