cri.TimeInput
时间输入框

属性

value Date

定义初始值

未设置时，默认使用当前系统时间

format String

Date 日期格式化字符串

默认值:yyyy/MM/dd HH:mm:ss

HMS Boolean

是否包括时分秒

当该值为false时，将忽略 format 时分秒部分

方法

enable

时间输入框可用

disable

时间输入框不可用

value

设置、获取时间输入框值

参数

value Date|String

函数接受/yyyy[/,-]MM[/,-]dd HH:mm:ss | yyyy[/,-]MM[/,-]dd格式的时间字符串

当value不为空，则该函数设置值，否则返回值

返回值

Date

当value不为空，则该函数设置值，否则返回值


            //取值
            $("#timeInput").data("timeInput").value();
            //设值
            $("#timeInput").data("timeInput").value(new Date());

getFormatValue

获取时间输入框格式化后的值

返回值

String

事件

change

当时间输入框值改变时触发

this 指向 timeInput 实例


        $("#timeInput").timeInput({
            change:function(){
                console.dir(this.value());//当时间输入框发生改变时，打印当前值
            }
        });
