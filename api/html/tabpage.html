cri.TabPage
TabPage分页组件

说明

该组件初始化时，读取直接子节点生成tab页

在每个子节点上获取data-title,data-close属性设置tab页标题和确定是否显示关闭按钮,当未设置data-close默认提供关闭按钮


        <div id="tabpage">
            <div data-title="tab1" data-close="false">
                |this is tab1
            </div>
            <div data-title="tab2">
                |this is tab2
            </div>
        </div>

事件

onFocus

当发生切换tab时触发

this 指向 tabPage 实例

事件数据

index

该事件向回调函数传入一个整形值，该值表示当前第几个Tab页被选中

该值从0开始


        $("#tabPage").tabPage({
            onFocus:function(index){
                var tabBody = this.getTabBody(index);//当选择某个Tab后,获取到ContentLoader对象
            }
        });

onCloseTab

当关闭某个Tab时触发

this 指向 tabPage 实例

事件数据

TabBody

该事件向回调函数传入一个ContentLoader对象


        $("#tabPage").tabPage({
            onCloseTab:function(tabBody){
                var content = tabBody.getContent();//获取tabBody中的内容的jQuery对象
            }
        });

方法

addNewTab

始终增加一个Tab,无论该Tab是否已经存在

参数

content String|jQuery

接受html字符串、jquery对象、URL用来生成TabBody内容

title String

在Tab上显示的标题

closeAble Boolean

是否在Tab上显示关闭按钮，默认值为显示关闭按钮

isIframe Boolean

当content为URL，将会根据isIframe参数决定是否使用iframe加载内容，否则使用jQuery.load方式加载内容

callBack Function

在生成TabBody后，回调该函数

返回值

ContentLoader

返回一个TabBody对象


        tabPage.addTab(
            content,  //tabPage 内容，可以传入 URL、jQuery、HTML字符串
            title,    //标题
            closeAble //在tabPage 标签头上是否提供关闭按钮
        );

addTab

增加Tab

参数

content String|jQuery

接受html字符串、jquery对象、URL用来生成TabBody内容

title String

在Tab上显示的标题

closeAble Boolean

是否在Tab上显示关闭按钮，默认值为显示关闭按钮

isIframe Boolean

当content为URL，将会根据isIframe参数决定是否使用iframe加载内容，否则使用jQuery.load方式加载内容

callBack Function

在生成TabBody后，回调该函数

返回值

ContentLoader

返回一个TabBody对象


        tabPage.addTab(
        content,  //tabPage 内容，可以传入 URL、jQuery、HTML字符串
        title,    //标题
        closeAble //在tabPage 标签头上是否提供关闭按钮
        );

closeTab

关闭第几个标签页

参数

index Number|String

接收一个整形，表示关闭第几个Tab,该值从0开始


        //参数为tab标签的索引值，索引值从0开始从左到右依次递增
        tabPage.closeTab(index);

select

选择第几个标签页

参数

index Number|String

接收一个整形，表示选择第几个Tab,该值从0开始


        //参数为tab标签的索引值，索引值从0开始从左到右依次递增
        tabPage.select(index);

getTabBody

根据索引获取TabBody

参数

index Number|String

接收一个整形，表示获取第几个TabBody,该值从0开始


        //参数为tab标签的索引值，索引值从0开始从左到右依次递增
        tabPage.getTabBody(index);
