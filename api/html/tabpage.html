cri.TabPage
TabPage分页组件

说明

该组件初始化时，读取直接子节点生成tab页，在每个子节点上获取data-title,data-close属性设置tab页标题和确定是否显示关闭按钮,当未设置data-close默认提供关闭按钮 


        <div id="tabpage">
            <div data-title="tab1" data-close="false">
                |this is tab1
            </div>
            <div data-title="tab2">
                |this is tab2
            </div>
        </div>

方法

addTab
void

增加Tab 


        tabPage.addTab(
            content,  //tabPage 内容，可以传入 URL、jQuery、HTML字符串
            title,    //标题
            closeAble //在tabPage 标签头上是否提供关闭按钮
        );

closeTab
void

关闭第几个标签页 


        //参数为tab标签的索引值，索引值从0开始从左到右依次递增
        tabPage.closeTab(index);

select
void

选择第几个标签页 


        //参数为tab标签的索引值，索引值从0开始从左到右依次递增
        tabPage.select(index);
