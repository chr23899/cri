cri.Tree
树形菜单组件

属性

title String

定义树形菜单题标

toolbar Array

定义树形菜单工具栏


        text:按钮文本
        iconCls:按钮图标,根据项目使用的字体库,cri默认依赖Font-Awesome字体库,请到Font-Awesome挑选图标
        handler:回调函数
        toolbar:[{text:'添加',iconCls:'fa fa-plus',handler:addFun},
            {text:'删除',iconCls:'fa fa-times',handler:deleteFun},
            {text:'修改',iconCls:'fa fa-pencil',handler:updateFun}
        ]

url String

数据请求地址

param Object

请求数据参数

async Boolean

是否展开节点时，请求节点数据,默认值:false

asyncUrl String

当async为true,通过该地址请求数据

方法

reload

根据参数重新请求数据并生成树

参数

param Object

请求参数，未指定时，使用上一次请求参数

getSelected

获取当前选择节点数据

返回值

Object

返回用户当前选择节点数据

事件

onSelected

当点击选中节点时，触发该事件

在方法中，this 指向 grid 实例

事件数据

selectData Object 


            $("#tt").tree({
                onSelected:function(selectData){
                    console.dir(selectData);//用户当前选择的节点数据
                }
            });

onDblClick

双击树形控件节点时触发

在方法中，this 指向 grid 实例

事件数据

selectData Object 


            $("#tt").tree({
                onDblClick:function(selectData){
                    console.dir(selectData);//用户双击,当前选择的节点数据
                }
            });
