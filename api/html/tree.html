cri.Tree
树形菜单组件

属性

title String

定义树形菜单题标

toolbar Array

定义树形菜单工具栏

url String

数据请求地址

param Object

请求数据参数

async Boolean

请求数据是否异步执行 true:异步 false：同步,默认值:false

asyncUrl String

当async为true,请求该地址请求数据

方法

reload

根据参数重新请求数据并生成树

getSelected

获取当前选择节点数据

事件

onSelected

当点击选中节点时，触发该事件

onDblClick

双击树形控件节点时触发
