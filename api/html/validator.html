cri.Validator
验证器

属性

rules Object

定义验证规则


        rules:{
            endDate:function($input){
                if($input.is('[name=endTime]')){
                    var endDate = $input.data("timeInput").value();
                    var startDate = $('.search-bar input[name=startTime]').data("timeInput").value();
                    if(startDate.getTime()>endDate.getTime()){
                        return false;
                    }
                }
                return true;
            },
            startDate:function($input){
                if($input.is('[name=startTime]')){
                    var startDate = $input.data("timeInput").value();
                    var endDate = $('.search-bar input[name=endTime]').data("timeInput").value();
                    if(startDate.getTime()>endDate.getTime()){
                        return false;
                    }
                }
                return true;
            }
        }

messages Object

定义rules对应的错误提示消息


    messages:{
        endDate:"结束时间不能小于结束时间",
        startDate:"开始时间不能大于结束时间"
    }

validateOnBlur Boolean

是否在输入框失去焦点时进行验证

方法

validate

进行验证

说明：请在标签上，通过添加属性，指定验证类型，内置的验证类型如下：


        1.required ([required])
        2.email ([role=email])
        3.url ([role=url])
        4.date ([role=date])
        5.pattern (在[role=text]|[role=email]|[role=url]|[role=tel]|[role=search]|[role=password]类型中根据[pattern]值进行正则验证)
        6.min(在[role=number]类型的输入框中根据[min='number']指定的值进行验证，当输入框中的值小于min指定的值，验证通过)
        7.max(在[role=number]类型的输入框中根据[max='number']指定的值进行验证，当输入框中的值大于max指定的值，验证通过)

hideMessages

隐藏显示的验证消息
