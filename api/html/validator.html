cri.Validator
验证器

属性

rules Object

定义验证规则


        rules:{
            endDate:function($input){
                if($input.is('[name=endTime]')){
                    var endDate = $input.data("timeInput").value();
                    var startDate = $('.search-bar input[name=startTime]').data("timeInput").value();
                    if(startDate.getTime()>endDate.getTime()){
                        return false;
                    }
                }
                return true;
            },
            startDate:function($input){
                if($input.is('[name=startTime]')){
                    var startDate = $input.data("timeInput").value();
                    var endDate = $('.search-bar input[name=endTime]').data("timeInput").value();
                    if(startDate.getTime()>endDate.getTime()){
                        return false;
                    }
                }
                return true;
            }
        }

messages Object

定义rules对应的错误提示消息


    messages:{
        endDate:"结束时间不能小于结束时间",
        startDate:"开始时间不能大于结束时间"
    }

validateOnBlur Boolean

是否在输入框失去焦点时进行验证

方法

validate

进行验证

说明：请在标签上，通过添加属性，指定验证类型，内置的验证类型如下：


        1.必填:required ([required])
        
        2.邮箱地址:email ([role=email])
        
        3.网络地址:url ([role=url])
        
        4.电话号码:tel ([role=tel])
        
        5.自定义正则表达式验证:pattern (在[role=text]类型中根据[pattern]值进行正则验证,可以通过[error-msg]指定该处的错误提示信息)
        
        6.最小值:min(在[role=range]|[role=number]类型的输入框中根据[min='number']指定的值进行验证，当输入框中的值小于min指定的值，验证通过)
        
        7.最大值:max(在[role=number]|[role=number]类型的输入框中根据[max='number']指定的值进行验证，当输入框中的值大于max指定的值，验证通过)
        
        8.数字输入:(在[role=number]类型的输入框中只能输入数字)

hideMessages

隐藏显示的验证消息

参数

names String|Array

当未传入参数表示隐藏所有消息

当传入一个字符串，表示隐藏对应name的输入框、选择框、时间框的验证消息

当传入一个数组，对数组中的每个项，隐藏对应name的输入框、选择框、时间框的验证消息
