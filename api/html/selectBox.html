cri.SelectBox
下拉选择框组件

属性

label

设置label,当未设置时,默认使用标签data-label,title,name属性值

value

设置初始值

未设置值，则使用原生元素value值

data

定义下拉框options


        data:[{value:"1",text:"1"},{value:"2",text:"2"}]

方法

enable

选择框可用

disable

选择框不可用

value

设置、获取选择框值

参数

value String|Number

当value不为空，则该函数设置selectBox值，否则返回selectBox当前选择项的值

返回值

String

当value不为空，则该函数设置selectBox值，否则返回selectBox当前选择项的值


            //取值
            $("select").data("selectBox").value();
            //设值
            $("select").data("selectBox").value("1");

设置、获取选择框文本值

参数

text String|Number

当text不为空，则该函数设置selectBox文本为text，否则返回selectBox当前选择项的文本值

返回值

String

当text不为空，则该函数设置selectBox文本为text，否则返回selectBox当前选择项的文本值


            //取值
            $("select").data("selectBox").text();
            //设值
            $("select").data("selectBox").text("A");

setOptions

重新设置selectBox下拉选项

参数

options Array

参数为数组，数组中的每一项为一个包含value、text字段的对象


            $("select").data("selectBox").setOptions([
                {value:1,text:'A'},
                {value:2,text:'B'},
                {value:3,text:'C'},
                {value:4,text:'D'}
            ]);

select

选择下拉选择框第几项

参数

index Number

从0开始的一个整形


            $("select").data("selectBox").select(0);//选择第一项

事件

change

当下拉框值改变时触发

this 指向 selectBox 实例


        $("select").selectBox({
            change:function(){
                console.dir(this.value());//当选择框选择项发生改变时，打印当前值
            }
        });
