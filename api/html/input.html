cri.Input
Input输入框

属性

label String

设置label,当未设置时,默认使用标签data-label,title,name属性值

value String|Number

初始化值

button Object

input右侧按钮 

        button={
            iconCls:"fa fa-times", //按钮图标
            handler:function(){   //点击按钮回调函数,this指向Input实例
                console.dir(this.value());
            }
        };

readonly Boolean

是否只读

方法

enable

输入框可用

disable

输入框不可用

value

设置、获取输入框值

参数

value String|Number

当value不为空，则该函数设置input值，否则返回input值

返回值

String

当value不为空，则该函数设置input值，否则返回input值


            //取值
            $("input").data("input").value();
            //设值
            $("input").data("input").value("hello");

事件

onFocus

当输入框获取焦点时触发

方法中 this 指向 input 实例


        $("#input").input({
            onFocus:function(){
                console.dir(this.value());
            }
        });

onBlur

当输入框失去焦点时触发

方法中 this 指向 input 实例


        $("#input").input({
            onBlur:function(){
                console.dir(this.value());
            }
        });
