cri.Window
窗口组件

属性

actions Array

定义窗口按钮,候选值:"Close","Minimize","Maximize"

模态窗口下(modal:true),忽略最小化action

content String|jQuery

接受html字符串、jquery对象、URL用来生成TabBody内容

isIframe Boolean

是否使用iframe,默认值:false

当content值为URL，根据isIframe判定是否通过iframe请求窗口内容，否则使用jQuery.load方式请求页面

visible Boolean

是否显示窗口,默认值:true

modal Boolean

是否初始化为模态窗口,默认值:false

width Number

定义窗口宽度,默认值:600

height Number

定义窗口高度,默认值:400

position Object

定义窗口位置,默认值:{top:0,left:0};

该值在center:false时起作用

center Boolean

居中显示窗口

该值为真时会屏蔽掉position设置

resizable Boolean

是否能缩放窗口,默认值：true

dragable Boolean

是否能拖动窗口,默认值：true

事件

onReady

当窗口完成构造时触发,this指向组件实例,并向回调函数传入窗口内容的Jquery对象

事件数据

content jQuery

窗口内容的Jquery对象


        $("#window").window({
            onReady:function($content){
                $content.find("input").input();//对窗口中所有input初始化
                this.open();//打开窗口
            }
        });

onOpen

当打开窗口时触发,this指向组件实例

onClose

当关闭窗口时触发,this指向组件实例

onMaximize

当最大化窗口时触发,this指向组件实例

onMinimize

当最小化窗口时触发,this指向组件实例

onResume

当复原窗口时触发,this指向组件实例

方法

load

加载窗口内容

参数

content String|jQuery|URL

接受HTML字符串，jQuery对象，URL

当参数为URL,将通过配置中的isIframe决定是否使用iframe加载内容，默认使用jQuery.load方式加载


        var win = $("#window").window({
            onReady:function($content){
                var iframe = $content.find('iframe');
                console.dir(iframe[0].id);//打印iframe的id
            }
        });
        win.load('http://www.channelsoft.com');

open

打开窗口

resume

还原窗口

minimize

最小化窗口

maximize

最大化窗口

close

关闭窗口

该方法将会销毁窗口实例

toFront

使窗口置为最前
