cri.Window
Window
窗口组件，继承自 Widgets

属性

actions
Array

定义窗口按钮,候选值:"Close","Minimize","Maximize",模态窗口下(modal:true),忽略最小化action

visible
Boolean

是否隐藏窗口,默认值:true

modal
Boolean

定义为模态窗口,默认值:false

width
Number

定义窗口宽度,默认值:600

height
Number

定义窗口高度,默认值:400

position
Object

定义窗口位置,默认值:{top:0,left:0};该值在center:false时起作用

center
Boolean

居中显示窗口,该值为真时会屏蔽掉position设置

resizable
Boolean

是否能缩放窗口

事件

onReady
Function

当窗口完成构造时触发,this指向组件实例,传入参数Window组件Jquery对象

方法

open
void

打开窗口

resume
void

还原窗口

minimize
void

最小化窗口

maximize
void

最大化窗口

close
void

关闭窗口
