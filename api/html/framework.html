cri.framework
framework
cri框架基础方法

方法

formValue
Object

获取或者设置表单值 


        //获取值
        var nameValues = $("form").formValue();

        //设置值
        $("form").formValue(nameValues);

isLeapYear
Boolean

判断闰年 


        var isLeapYear = cri.isLeapYear(new Date());

formatDate
String

转换日期成字符串 


        var dateStr = cri.formatDate(new Date());

string2Date
Date

字符串转换成Date 


        var date = cri.string2Date('2014/06/07');

date2Json
Object

Date转换成对象{yyyy:'2015',MM:'01',dd:'01',mm:'00',HH:'00',ss:'00',ww:'4'} 


        var date = cri.date2Json('2015/01/01');

date2Array
Array

Date保存到数组[年,月,日,时,分,秒] 


        var arr = cri.date2Array(new Date());
