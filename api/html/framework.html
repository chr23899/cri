cri.framework
cri框架基础方法

jQuery 拓展方法

formValue

获取或者设置表单值


        //获取值
        var nameValues = $("form").formValue();
        //设置值
        $("form").formValue(nameValues);

formReset

重置表单

该方法可接收一个对象，用来指定某些字段的默认值

参数

data


        $("form").formReset({
            fieldA:'abc',
            fieldB:123
        });

日期类工具函数

isLeapYear

判断闰年

参数

year Number|String

整数年份,或者字符串

返回值

Boolean 


        var isLeapYear = cri.isLeapYear(new Date());

formatDate

将日期转换成指定日期格式的字符串

参数

date Date

准备转化的日期对象

formatStr String

日期格式串,未指定该值时,默认使用 'yyyy-MM-dd' 格式串

YYYY/yyyy/YY/yy 表示年份

MM/M 月份

W/w 星期

dd/DD/d/D 日期

hh/HH/h/H 时间

mm/m 分钟

ss/SS/s/S 秒

返回值

String

格式化后的日期字符串


        var dateStr = cri.formatDate(new Date(),'yyyy-MM-dd HH:mm:ss');

string2Date

字符串转换成Date

参数

dateStr String

日期字符串,日期格式目前只接收 YYYY/MM/dd HH:mm:ss YYYY-MM-dd HH:mm:ss YYYY/MM/dd YYYY-MM-dd

返回值

Date

日期对象


        var date = cri.string2Date('2014/06/07');

验证工具函数

isArray

判断是否为数组

参数

o Array|Object|String|Number

返回值

Boolean

当参数为数组时，返回真


        cri.isArray([1,12,3]);

isNum

判断是否为数组

参数

o Array|Object|String|Number

返回值

Boolean

当参数为数字时，返回真


        cri.isNum('123');

isPhoneNo

判断是否有效电话号码

参数

phoneNo String

返回值

Boolean

当参数为有效电话时，返回真


        cri.isPhoneNo('1801234567');
